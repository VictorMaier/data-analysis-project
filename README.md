# Universal Statistics System

Система для загрузки, очистки, статистического анализа и визуализации данных CSV.
Разработана в рамках учебного курса.

## Функционал
- [x] Загрузка данных (CSV)
- [x] Очистка данных (удаление дубликатов, пропусков, замена на среднее)
- [x] Статистический анализ (Мода, Медиана, Дисперсия, Корреляция)
- [x] Визуализация (Гистограммы, BoxPlot)
- [x] Машинное обучение (Линейная регрессия)

## Установка и запуск

1. Клонируйте репозиторий:
   git clone https://github.com/VictorMaier/data-analysis-project.git

2. Создайте виртуальное окружение и установите зависимости:
   python -m venv venv
   pip install -r requirements.txt

3. Запустите программу:
   python main.py

## Используемые технологии
- Python 3.14
- Pandas, NumPy (Обработка данных)
- Matplotlib, Seaborn (Визуализация)
- Scikit-learn (ML)
